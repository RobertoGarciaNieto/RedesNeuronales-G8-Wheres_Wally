# Redes Neuronales Profundas - TP Integrador: Where's Waldo?
Este proyecto implementa una red neuronal para la detección de personajes en el dataset "Where's Waldo".

## Estructura del repositorio
- `/data`: Contiene los archivos CSV con los splits (train, val, test).
- `/dev`: Contiene los notebooks de desarrollo.
- `/prod`: Carpeta reservada para el modelo entrenado y el script de inferencia.

## Dataset
- **Fuente:** [Kaggle - Where's Waldo](https://www.kaggle.com/datasets/mohaneddz/wheres-waldo)
- **Instrucciones:** Las imágenes no se versionan en este repositorio por cuestiones de tamaño. El notebook `dev/01_dataset_preparation.ipynb` descarga y descomprime el dataset automáticamente al ejecutarse.
